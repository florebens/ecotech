# EcoTech

The Materiality of Higher Civilization Engineering.

A curated database of ecological technologies, grassroots inventions, and 
regenerative building practices from around the world — with emphasis on 
low-cost, high-impact solutions built outside institutional frameworks. 

This database values passive energy solutions, vernacular architecture, self-sustaining systems, elegant intervention.

## Categories

- **Materials** — creation, recycling, repurposing
- **Structures** — architecture, engineering, building systems
- **Inventions** — grassroots creative force
- **Community** — policy, activism, collective solutions
- **Events** — courses, meetings, newsworthy milestones
